# Support

## Getting help

If you need help using Kinetik:

- Open a GitHub Discussion: https://github.com/sumitttt4/Kinetik/discussions
- Open an issue for reproducible bugs: https://github.com/sumitttt4/Kinetik/issues

## Security issues

Please **do not** report security vulnerabilities publicly.

Use the process in [SECURITY.md](SECURITY.md) and email:
- sumitsharma.dev@proton.me

## Sponsorship

If Kinetik helps your team, you can support maintenance at:
- https://buymeacoffee.com/sumitsharmq

## Response expectations

- Issues/discussions: best effort, typically within a few days
- Security reports: triaged within 72 hours
