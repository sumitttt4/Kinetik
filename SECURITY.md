# Security Policy

## Supported Versions

Kinetik is currently in active development. We provide security updates for the latest commit on `main`.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Please report it privately by email:
- **Email:** sumitsharma.dev@proton.me
- **Subject:** `[Kinetik Security] <short summary>`

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested remediation (if known)

## Response Timeline

- Initial response: within **72 hours**
- Triage decision: within **7 days**
- Fix and disclosure timeline: based on severity and exploitability

We follow coordinated disclosure and will credit reporters (if desired) after patches are released.
